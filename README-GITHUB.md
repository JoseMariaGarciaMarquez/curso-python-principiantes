## 📖 Descripción

Este repositorio contiene un curso completo de **introducción a Python** diseñado para principiantes que quieren aprender programación desde cero. El material incluye tanto lecciones teóricas como notebooks interactivos para una experiencia de aprendizaje completa.

## ✨ Características

- 📚 **Material teórico completo** con explicaciones claras
- 📓 **Notebooks interactivos** para práctica inmediata
- 🏋️ **Ejercicios progresivos** desde básico hasta avanzado
- 🎯 **Proyectos prácticos** para aplicar conocimientos
- 🛠️ **Configuración paso a paso** para cualquier sistema operativo
- 🎨 **Arte ASCII y creatividad** para hacer el aprendizaje divertido

## 🎯 ¿Para quién es este curso?

✅ **Principiantes absolutos** en programación  
✅ **Estudiantes** que quieren un recurso estructurado  
✅ **Autodidactas** que prefieren aprender a su ritmo  
✅ **Educadores** que buscan material de calidad  
✅ **Cualquier persona** curiosa por aprender Python

## 🚀 Inicio Rápido

### 1. Clonar el Repositorio
```bash
git clone https://github.com/TU_USUARIO/curso-python-principiantes.git
cd curso-python-principiantes
```

### 2. Verificar Python
```bash
python --version
# o
python3 --version
```

### 3. Abrir el Material
```bash
# Para notebooks (recomendado)
jupyter notebook

# Para VS Code
code .
```

### 4. Empezar a Aprender
1. Lee el [README principal](./README.md)
2. Sigue la [guía de instalación](./INSTALACION.md)
3. Comienza con [tu primer programa](./01-preparacion/primer-programa.md)
4. Practica con los [notebooks interactivos](./notebooks/)

## 📖 Cómo Usar Este Curso

### 📱 Opción 1: VS Code (Recomendado)
1. Instala [VS Code](https://code.visualstudio.com/)
2. Instala la extensión de Python
3. Abre la carpeta del curso
4. Sigue las lecciones en orden

```bash

jupyter notebook
```
